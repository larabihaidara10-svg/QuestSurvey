# QuestSurvey - User Guide

## Introduction

QuestSurvey is a web-based questionnaire system for creating, managing, and analyzing surveys. It allows creators to build surveys with various question types and collect responses from participants.

---

## Getting Started

### 1. Installation

1. Upload all files to your web server
2. Navigate to the application URL
3. Create your admin account (first name, last name, username, password)
4. Login with your credentials

### 2. Creating a Survey

1. Click **"Create"** or **"Create Survey"**
2. Enter survey title and description
3. Select **Identity Mode**:
   - **Anonymous**: No identification required (for course evaluation)
   - **Identified**: Name and student number required
   - **Access Code**: One-time codes for limited responses
4. Set availability dates (optional)
5. Click **"Create Survey"**

### 3. Adding Questions

1. From "My Surveys", click **"Edit"** on your survey
2. Select question type:
   - Multiple Choice
   - Likert Scale (1-5)
   - Short Text
   - Long Text
   - Yes/No
   - Date
   - Number
3. Enter question text
4. For Multiple Choice, add options
5. Mark as required if needed
6. Click **"Add Question"**

### 4. Managing Surveys

- **Edit**: Modify survey title, description, or questions
- **View**: Preview or take the survey
- **Results**: View analysis and export data
- **Delete**: Remove the survey

---

## Taking a Survey

### Anonymous Mode
- No identification required
- Simply fill in answers and submit

### Identified Mode
- Enter your name and student number
- Required before answering questions

### Access Code Mode
- Enter the unique code provided by the creator
- Code can only be used once
- After submission, the code is invalidated

---

## Viewing Results

1. Go to "My Surveys"
2. Click **"Results"** on any survey

### Analysis Features:
- **Multiple Choice**: Bar chart + count/percentage table
- **Likert Scale**: Pie chart + average score + distribution
- **Yes/No**: Response counts
- **Number**: Average, min, max
- **Text**: List of all responses

### Export Data
- Click **"Export CSV"** to download all responses

---

## Access Code Management

When creating a survey with "Access Code" mode:
1. Specify number of codes needed (e.g., 30 for 30 students)
2. System generates unique codes
3. Print/copy codes for distribution
4. Each code can only be used once
